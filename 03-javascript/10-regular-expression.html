정규표현식(Regular Expressiong)

문자열에서 특정 패턴을 찾거나, 대체하거나, 나누는 도구

1. 자주 쓰는 메타문자

  • ^ : 문자열의 시작
  • $ : 문자열의 끝
  • . : 임의의 한 문자
  • * : 앞 문자가 0회 이상
  • + : 앞 문자가 1회 이상
  • ? : 앞 문자가 0회에서 1회
  • {m,n} : m ~ n회 반복
  • [] : 문자셋
  • () : 그룹, | : 또는
  • \d : 숫자 (0-9), \D : 숫자 제외
  • \w : 문자/숫자/_, \W : 문자/숫자/_ 제외
  • \s : 공백, \S : 공백 제외

2. match / replace / split

예시) 주민등록번호를 정규표현식으로 작성
