# GUDDY 커뮤니티 채팅

실시간 멀티유저 채팅 서버 (Node.js + Socket.io)

## 실행 방법

### 1. 의존성 설치
```bash
npm install
```

### 2. 서버 시작
```bash
node server.js
```

브라우저에서 http://localhost:3000 접속하면 됩니다.

---

## 외부에서 접속하게 하는 방법

### 방법 A: 같은 Wi-Fi 안에서 공유 (간단)
1. 서버 실행 후 내 IP 확인
   - Windows: `ipconfig` → IPv4 주소 확인
   - Mac/Linux: `ifconfig` 또는 `ip addr`
2. 같은 Wi-Fi 친구에게 `http://192.168.x.x:3000` 주소 공유

### 방법 B: 인터넷 어디서나 접속 (ngrok 사용)
1. https://ngrok.com 에서 ngrok 설치
2. 서버 켜놓고 새 터미널에서:
   ```bash
   ngrok http 3000
   ```
3. 출력된 `https://xxxx.ngrok.io` 주소를 친구들에게 공유
